# Vanilla-Nemesis
Vanilla Nemesis - 16-bit homebrew CPU with custom ISA. Hardwired control, static registers (rd0, rw0, rw1), cloud stack with random access (no LIFO), built-in interrupt controller, transparent debugging via dp instruction. No x86 legacy. No Real Mode. Starts directly in 2026. Built in Logisim from scratch by a 13-year-old developer. pwoff and relax
